# Changelog: Kognitika (Когнитика)

## [1.2.0] - 2026-05-12
### Добавлено
- **Вращение цифр**: Новая экспериментальная опция в Таблицах Шульте для тренировки вестибулярной устойчивости и концентрации.
- **Настройка скорости моргания**: Слайдер в настройках позволяет регулировать частоту визуальных помех (Chaos) от 100мс до 3000мс.
- **Динамический уровень 4**: Теперь включает автоматическое вращение цифр для максимальной сложности.

### Изменено
- **Безопасность UI**: Удалены прямые ссылки на админ-панель из всех меню. Доступ теперь только по прямому адресу `/admin`.
- **Оптимизация визуальных эффектов**: Эффект "Chaos" переведен на детерминированный временной цикл, что устраняет "дергание" и снижает риск возникновения тошноты при прохождении.
### Исправлено
- **Стабилизация мобильного интерфейса**: Устранен критический баг «белого экрана» (White Screen of Death) при открытии бокового меню на мобильных устройствах.
- **Исправление иконок**: Исправлены битые импорты `lucide-react` (в частности `Lightbulb`), вызывавшие ошибки во время выполнения.
- **Интеграция разделов**: Восстановлена корректная работа раздела «Предложения» (IdeasWall), исправлена привязка в навигации.
- **Синхронизация с VPS**: Проведен полный цикл деплоя и пересборки приложения на сервере `kognitika.syntog.ru`.

### Технические улучшения
- Рефакторинг `generateChaosStyle` для поддержки синхронизированных по времени анимаций.
- Обновление `useSchulteEngine` для поддержки расширенных настроек модификаторов.

## [1.1.0] - 2026-05-11
### Добавлено
- Модуль "Модифицированные таблицы Шульте" с 4 уровнями сложности.
- Система опыта (XP) и уровней атлетов.
- Глобальный лидерборд (Зал Славы).
- Аудио-помехи (голосовое сбивание счета).

### Изменено
- Обновлен дизайн главной страницы в стиле "Dark Glassmorphism".
- Интеграция с Prisma DB для сохранения игровых сессий.
